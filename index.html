CanvasGear 0.1.2

Slogan: CanvasGear is a JavaScript program to draw animated canvases by just specifying some parameters.

Quickstart: Copy canvasgear.js and canvasgearexcanvas.js to your pages folder. Then place the following outlined code in your HTML file: 

 <html>
  <head>
   <script src="./canvasgear.js"></script>           <!-- (1) load script -->
   <!--[if lte IE 9]><script src="./canvasgearexcanvas.js"></script><![endif]--> <!-- (2) load IE compatibility -->
  </head>
  <body onload="startCanvasGear();">                <!-- (3) start work -->
   <canvas id="myCanvas44" width="123" height="123"></canvas> <!-- (4) define a canvas -->
   <!-- algo=pulse color=hotpink hertz=0.2 -->      <!-- (5) specify parameters for canvas -->
  </body>
 </html>
