Installation

(1)   Quick Installation

(1.1)   Copy the two files canvasgear.js and canvasgearexcanvas.js to your document folder.

(1.2)   In your HTML file, provide the five ingredients as outlined below. 

 <html>
  <head>
   <script src="./canvasgear.js"></script>         <!-- (1) load script -->
   <!--[if lte IE 9]><script src="./canvasgearexcanvas.js"></script><![endif]--> <!-- (2) load IE helper script -->
  </head>
  <body onload="startCanvasGear();">               <!-- (3) start script -->
   <p>
    <canvas id="myCanvas44" width="222" height="222"></canvas> <!-- (4) define canvas -->
    <!-- algo=algo1 color=yellow hertz=0.3 -->     <!-- (5) specify parameters -->
   </p>
  </body>
 </html>

Ready. Now you should see in your HTML page the defined canvases, possibly animated.

About wich parameters exactly are available, please see (the source of) e.g. the Demo page.

About how many canvases you can place on one page, see page 'Features' subchapter Performance .

(2)   Big Installation

(2.1)   Copy the complete CanvasGear folder to your document folder.

(2.2)   In your HTML files script tag, for the script path, use the relative path to the CanvasGear folder.

(2.3)   About the body tag and the canvas tag, follow the steps described above.

You can surfe this CanvasGear site locally now.

By the way, the site works not only from a webserver, but also with the file protocol 'file://'. This is convenient if no webserver is wanted, e.g. when viewing the site from an USB drive.

That's it.

Here some explanation, what above pointed out lines mean in detail :

  • <script src="./canvasgear.js"></script>
    In the header, the CanvasGear script loaded.
  • <!--[if lte IE 9]><script src="./canvasgearexcanvas.js></script><![endif]-->
    In the header, the IE compatibility script is conditionally loaded if the browser is IE8 or older.
  • <body onload="startCanvasGear();">
    In the body tag, the onload function is registered. This can be done together with possible other functions.
  • <canvas id="myCanvas44" width="222" height="222"></canvas>
    Somewhere in the body, a canvas is placed with an unique id.
  • <!-- algo=pulse color=yellow hertz=1 -->
    The comment elelment immediately behind the canvas holds a commandline specifying the drawing parameters.
